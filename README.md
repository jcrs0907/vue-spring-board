# vue-spring-board

## 개발 환경 구성
JAVA 11, Spring Boot, JPA, Vue.js, h2 Database

## h2 Database 설정
인메모리, MySql과 같이 동작하게 구성중..

## 프론트 개발 환경
포트 번호: 3000
> vue 폴더에서 npm run dev 
> localhost:3000 접속

## 백앤드 개발 환경
배포할 때 npm build가 자동으로 수행되도록 설정되어 있다.
포트 번호: 8080
